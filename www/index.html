No.1 goal? Maximize productivity!

Create a static website, web and mobile apps with CORS authentication that share assets connected to micro-service backend in around 20 mins plus the time to have a coffee while npm install does its thing.

With StackMates you can manage the digital strategy of multple domain to quickly launch experiments from a central system based on Gulpjs, Browserify, your choice of Rework or Sass.

Metalsmith combines with Google Drive to create to create a static content and project management system.

Angularjs and Ionic framework combine to create Line of Business and Mobile apps respectively which use a json file created as a side effect of running Metalsmith to enable the reuse static content for prototyping, or use with basic apps.

On the backend Hapijs combines with Senecajs to create a three tiered architecture that can be run as a single app or broken into a micro-services based on Docker.

Learn

The number one goal of StackMates is to maximise productivity. Because there are a lot of new concepts presented here, fast absorption of knowledge is key to achieving this goal. So the intent is that each major concept of the design will have associated Mind Maps and ANKI flash cards to help you grasp and remember concepts and conventions so you can be as productive as possible.
